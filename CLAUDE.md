# 採用LP - Claude Code Project Memory

## プロジェクト概要

Studio Altea（みちがえる）のpixiv AD向け採用LP。
成人向けASMR音声制作スタッフ（音声編集・SNS運用）を募集。

## 技術構成

- **形式**: 単一HTML（静的ホスティング用）
- **CSS**: Tailwind CSS（CDN）
- **フォント**: Zen Kaku Gothic New + Noto Sans JP（Google Fonts）
- **アイコン**: Lucide Icons（CDN）
- **ホスティング**: GitHub Pages
- **ドメイン**: studio-altea.jp（CNAME設定済み）

## フォルダ構成

```
recruitment-lp/
├── index.html         # LP本体（メインファイル）
├── CNAME              # GitHub Pages用ドメイン設定
├── README.md          # リポジトリ説明
├── CLAUDE.md          # CC用プロジェクトメモリ（このファイル）
├── .claude/           # CC設定
│   └── settings.json  # プロジェクト固有設定
└── assets/
    └── images/        # 作品サムネイル画像
```

## デザイン方針


### 避けるべきこと（AI Slop）

- ❌ 紫グラデーション on 白背景
- ❌ 全部センター揃え
- ❌ Inter/Roboto/Arial系フォント
- ❌ 均等な角丸（rounded-lg統一）
- ❌ 予測可能なグリッド配置

### 目指すこと

- ✅ ダークモード基調で同人・クリエイター文化に寄せる
- ✅ 非対称レイアウト、意図的な余白
- ✅ 蛍光カラーでサイバー感を演出
- ✅ 作品画像を効果的に見せる

## 募集要項

### 音声編集スタッフ（2名）
- 業務: ASMR音声の編集、SE挿入、マスタリング
- 条件: DAWを触ったことがある（趣味レベルOK）
- 歓迎: バンド経験、宅録経験、同人音声リスナー

### SNS運用スタッフ（1名）
- 業務: X運用、DLsite告知、ファンコミュニケーション
- 条件: ASMR・同人文化に興味がある
- 歓迎: X運用経験、推し活経験

### 共通条件
- 雇用形態: 正社員
- 勤務地: 秋葉原オフィス（出社）
- 給与: 月給25万円〜
- 必須: 18歳以上、アダルトコンテンツに抵抗がない
